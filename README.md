# 🚀 HyperOS 3 Pack ⟨Phone Version⟩ - Magisk Module

![Magisk](https://img.shields.io/badge/Magisk-Module-orange?style=for-the-badge&logo=android)
![Version](https://img.shields.io/badge/Version-v12-blue?style=for-the-badge)

Ubah tampilan HyperOS 2 kamu menjadi gaya **HyperOS 3** yang lebih modern dan fresh! Modul ini dirancang khusus untuk memberikan pengalaman visual terbaru pada perangkat Xiaomi Anda.

## ✨ Fitur Utama
* 📱 **New Status Bar:** Tampilan status bar khas HyperOS 3.
* 🎨 **Redesigned Icons:** Icon pack yang lebih halus dan konsisten.
* ⚙️ **Settings UI:** Perubahan ikon di dalam menu Setelan (Settings) ke gaya HyperOS 3.
* 🔄 **Auto-Update Support:** Update langsung lewat aplikasi Magisk tanpa ribet!
* ✅ **Pemilih via volume:** pilih bagian mana saja yang ingin di pasang Magisk tanpa ribet!

## 🛠️ Persyaratan
- Perangkat Xiaomi/Redmi/Poco.
- Sudah terpasang **Magisk** atau **Kitsune Mask**.
- Berjalan di basis HyperOS 2.

## 📥 Cara Instalasi
1. Download file `.zip` versi terbaru dari [sini](https://github.com/yasin3unnity/HyperOS-3-Pack-2/raw/refs/heads/main/HOS3_Pack_11_For_Phone.zip).
2. Buka aplikasi **Magisk**.
3. Masuk ke tab **Modules** lalu klik **Install from storage**.
4. Pilih file yang sudah di-download tadi.
5. Reboot perangkat Anda.
